# KICEasy
평가원식 그래프 계산기

## Supported OS
- Windows 10 or higher

## Usage
Start typing a math expression first, then click f(x) icon to display graph

## Function example
```
y=x^2-3x+1
```

## Build requirements
- Visual Studio 2022 or higher
- MFC v143 or higher
